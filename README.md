# 🤖 بوت حماية المجموعات — واتساب

بوت تلقائي يحذف الرسائل الإعلانية ويطرد أصحابها فوراً.

---

## ⚙️ طريقة التثبيت

### المتطلبات
- Node.js الإصدار 16 أو أحدث
- جهاز كمبيوتر أو سيرفر يعمل 24/7
- رقم واتساب مخصص للبوت (غير رقمك الشخصي)

### خطوات التثبيت

```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل البوت
npm start
```

### ربط الواتساب
بعد التشغيل ستظهر لك **صورة QR Code** في الشاشة.
افتح واتساب على جوالك ← الإعدادات ← الأجهزة المرتبطة ← امسح الكود.

---

## 🚫 ما يقوم به البوت تلقائياً

| المخالفة | الإجراء |
|----------|---------|
| إرسال رقم جوال (9 أو 10 أرقام) | حذف الرسالة + طرد فوري |
| رقم يبدأ بـ +966 أو +967 | حذف الرسالة + طرد فوري |
| أي رقم دولي +XXXX | حذف الرسالة + طرد فوري |
| كلمة ممنوعة | حذف الرسالة + طرد فوري |

---

## 📋 أوامر الأدمن (داخل المجموعة)

| الأمر | الوظيفة |
|-------|---------|
| `!اضف خصوصي` | إضافة كلمة ممنوعة جديدة |
| `!احذف خصوصي` | حذف كلمة من القائمة |
| `!الكلمات` | عرض جميع الكلمات الممنوعة |
| `!مساعدة` | عرض قائمة الأوامر |

> ⚠️ الأوامر تعمل فقط من رقم صاحب البوت (الرقم المرتبط بالجهاز)

---

## ⚠️ تنبيه مهم

هذا البوت يعمل عبر مكتبة غير رسمية.
- استخدم رقماً مخصصاً للبوت وليس رقمك الشخصي
- واتساب قد يحظر الرقم عند الإفراط في الاستخدام
- يُنصح بتشغيله على سيرفر ثابت
